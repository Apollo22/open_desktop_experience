# Desktop Environment

A desktop environment is a collection of softwares that provide a __desktop environment user experience__. There are composed of the following types of softwares :

* Windows display
** Windows Manager
** Windows Compositor
** Workspace / Virtual desktops manager / pager
** Taskbars / Toolbars
*** Applications
*** Status
** Wallpaper manager
*** Widgets
 
* Notifications
* 
** Login Screen / Display Manager
** Logout / Shutdown screen
** Screensaver

* Settings softwares
** Setting software
*** Languages
*** I/O Manager
** Clipboard manager
** Sound manager
** Network manager
** Power manager

* Specific softwares

** File manager
** Application launcher

** Image Viewer
** Calculator
